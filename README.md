# WL-Flavors

Projeto white label para praticar criação de Product Flavors.

## <br />Tecnologias utilizadas
1. MVVM e Clean Architecture
2. Material Design
3. Firebase (storage e firestore)
4. Coroutines
5. Hilt
6. Navigation
7. Glide
